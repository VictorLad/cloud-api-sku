# Cloud Api Sku

## Install project

You already have installed:
    - [nodejs](https://nodejs.org/en/)

Then, launch these commands in the root project directory

```
npm install
```

## Command lines

```

### Start application

```
npm start
```

### Run tests

```
npm run test
```

### Deploy snapshot

```
npm run deploy
```

### Release

```
npm run release
```
